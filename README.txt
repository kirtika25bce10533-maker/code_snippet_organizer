Code Snippet Organiser

Introduction

The Code Snippet Organiser is a Python-based project designed to store
and retrieve commonly used code snippets. It simplifies coding workflow
by allowing users to save snippet titles with their code and access them
whenever needed.

Features

-   Add a new code snippet
-   View all saved snippets
-   Search snippet by title
-   Delete a snippet
-   Text-file based storage

Technologies Used

-   Python
-   File Handling (snippets.txt)

How It Works

The application provides a menu: 1. Add Snippet
2. View All Snippets
3. Search Snippet
4. Delete Snippet
5. Exit

Snippets are stored in snippets.txt using the format:

    title:::snippet

How to Run

1.  Install Python
2.  Save the script as snippetorganiser.py
3.  Run using:

    python snippetorganiser.py

Conclusion

A helpful tool for students and programmers to store frequently used
code. This project demonstrates file handling, loops, and user-driven
menus.
